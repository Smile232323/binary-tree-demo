# Welcome — 项目笔记

这是你的项目笔记示例，使用 Markdown All in One 插件可在 VS Code 中实时预览和编辑。

## 快速上手
- 打开：在资源管理器中新建文件 `notes/Welcome.md`（已为你创建）。
- 预览：按 `Cmd+Shift+V` 或打开命令面板运行 `Markdown: Open Preview`。
- 并排编辑与预览：按 `Cmd+K` 然后 `V`。
